YOLOv5 Traffic Intersection Analysis

In this project, I was asked to make a program that could count and analyse the objects passing through an intersection. We had gone through many ideas before settling with using yolov5 object detection and strong sort object tracking to keep track of objects. Using this implementation, we were able to keep track of the labels of each object, and have relatively accurate pixel coordinates.

To analyse these intersions, we needed to process the outputs from the yolov5 program. Using multiprocessing, we were able to determine the cars Time to Collision and Post Encroachment Time.
